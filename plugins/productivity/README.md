# 效率插件

提升日常开发效率的全方位工具集，让开发工作更快、更流畅。

## 功能特性

### 🚀 快速脚手架
- 一键生成项目结构
- 支持多种项目模板
- 自定义模板配置

### 📝 文档自动生成
- 根据代码自动生成文档
- 支持多种文档格式（Markdown、JSDoc、TSDoc）
- API 文档自动提取

### 🔧 智能重构
- 代码结构优化建议
- 自动提取函数/组件
- 变量和函数智能重命名
- 代码去重和简化

### 📦 代码片段管理
- 保存常用代码片段
- 团队共享代码片段
- 智能代码模板

### 🔄 批量操作
- 批量重命名文件和变量
- 批量导入/导出处理
- 批量格式化

## 使用方法

### 快速生成脚手架
```bash
/quick-scaffold react-component
/quick-scaffold api-endpoint
/quick-scaffold test-suite
```

### 生成文档
```bash
/generate-docs
```

### 代码重构
```bash
/refactor-assist
```

### 管理代码片段
```bash
/snippet-manager
```

### 批量重命名
```bash
/batch-rename
```

## 安装

```bash
/plugin marketplace add jianlanglinhei/vibe-plugins
/plugin install productivity
```
