# 稳定性插件

确保代码质量和系统稳定性的综合工具集。

## 功能特性

### 🛡️ 错误检查
- 全面的错误检测
- 异常处理建议
- 边界条件检查
- Null/Undefined 检查

### ⚡ 性能监控
- 性能瓶颈分析
- 代码性能评分
- 优化建议
- 资源使用分析

### 🧪 测试覆盖
- 测试覆盖率统计
- 缺失测试识别
- 单元测试生成
- 集成测试建议

### 💾 内存管理
- 内存泄漏检测
- 循环引用识别
- 资源释放检查
- 性能优化建议

### 🔒 安全扫描
- 依赖漏洞检查
- SQL 注入检测
- XSS 防护检查
- 敏感信息扫描

## 使用方法

### 错误检查
```bash
/error-check
```

### 性能分析
```bash
/performance-audit
```

### 测试覆盖率
```bash
/test-coverage
```

### 内存泄漏检测
```bash
/memory-leak
```

### 安全扫描
```bash
/security-scan
```

## 自动化 Hooks

插件会在以下时机自动运行检查：
- 代码提交前
- Pull Request 创建时
- 部署前检查

## 安装

```bash
/plugin marketplace add jianlanglinhei/vibe-plugins
/plugin install stability
```
