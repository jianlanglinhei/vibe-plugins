# /analyze - AI 工具能力分析

深度分析项目中的 AI 工具配置，生成结构化能力清单报告。

## 使用方法

```
/analyze                      # 分析当前项目的 AI 工具能力
/analyze <path>               # 分析指定目录
```

## 分析维度

1. **能力类型识别**
   - Prompt - 提示词模板
   - Skill - 技能配置
   - MCP - MCP 服务器工具
   - Slash Command - 斜杠命令
   - Agent - 智能体配置
   - Hook - 钩子脚本

2. **分类归属**
   - 体验优化
   - 一码多端
   - 其他自定义分类

3. **工具描述**
   - 工具的功能说明和用途

## 输出格式

```
📊 AI 工具能力分析

| 能力类型      | 分类归属 | 工具描述                           |
|---------------|----------|-----------------------------------|
| Prompt        | 体验优化 | 大屏适配助手 - 自动适配大屏布局      |
| Prompt        | 体验优化 | 无障碍适配助手 - 生成无障碍属性      |
| Skill         | 一码多端 | Weex to 小程序转换器               |
| MCP           | 体验优化 | 无感加载配置器 - 配置预加载策略      |
| MCP           | 一码多端 | 一键提测工具 - 自动生成提测单        |
| Slash Command | 一码多端 | 变更分支自动化集成工具              |
| Slash Command | 一码多端 | bug 自动修复工具                   |
| Agent         | 一码多端 | 一键提取网页 markdown              |
| Agent         | 一码多端 | Orisa 转码 Agent                  |
```
